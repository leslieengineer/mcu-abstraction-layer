# 99 — Archive

> Deprecated, superseded, or no-longer-applicable documents.
> **Never delete** project documents — move them here.

## Rules

1. When moving a document here, record the move in `CHANGELOG.md`
2. Update `Official-Document-Register.md` — set status to `Archived`
3. Add a note at the top of the archived file:

```markdown
> **ARCHIVED** — Superseded by {{NEW_DOCUMENT}} on {{DATE}}.
> Reason: {{REASON}}
```

4. Do NOT modify archived documents after moving them here
