# 10 — Operations & Support

> Post-deployment support: diagnostics, monitoring, and field troubleshooting.

**PX4 v1.16 context**: PX4 uses ULog binary logging, MAVLink `STATUSTEXT` for real-time alerts, NSH console for interactive debugging, and `hardfault_log` for post-crash analysis. QGroundControl provides the primary operator interface.
